# 15. How-To Guides

## How to Connect a Social Media Platform
1. Go to **Platforms** page
2. Find the platform you want to connect
3. Click **Connect**
4. Enter your API Key, API Secret, Access Token, and Access Token Secret
5. Click **Save** — the platform will show as "Connected"

## How to Schedule a Post
1. Navigate to **Content Planner**
2. Click **Schedule Post** or click a date on the calendar
3. Write your content (or use AI suggestions)
4. Select one or more target platforms
5. Set the date and time
6. Optionally attach media
7. Click **Schedule** to queue the post

## How to Use the AI Content Creator
1. Go to **AI Content** page
2. Enter a topic or describe what you want to post about
3. Select the target platform for tone optimization
4. Click **Generate** to get AI-powered suggestions
5. Edit the suggestion as needed
6. Copy to clipboard or send directly to Content Planner

## How to Set Up Webhooks
1. Navigate to **API Integrations** → **Webhooks** tab
2. Click **Add Webhook**
3. Enter a name and the callback URL
4. Select which events to subscribe to
5. Click **Create**
6. Use **Test** to send a sample event

## How to Export Analytics Data
1. Go to **API Integrations** → **Data Import/Export** tab
2. Select data type (posts, analytics, platforms)
3. Choose format (CSV, JSON, XLSX)
4. Set date range if applicable
5. Click **Export** — file downloads automatically

## How to Import Data
1. Go to **API Integrations** → **Data Import/Export** tab
2. Click **Import**
3. Download the template for your data type
4. Fill in the template with your data
5. Upload the file
6. Review the import summary

## How to View Audience Insights
1. Navigate to **Audience Insights**
2. Select a platform from the dropdown
3. Review demographics (age, gender, location)
4. Check engagement patterns by day of week
5. See best posting times and top hashtags

## How to Update Your Profile
1. Click your avatar in the top-right corner
2. Select **Profile**
3. Edit your name, bio, website, and location
4. Click **Save Changes**
